## 📌 Descrição do Projeto

Este projeto consiste em um **Sistema de Gestão de Compras**, desenvolvido utilizando o framework **Vue.js** com a Composition API.

O objetivo é aplicar conceitos fundamentais de frameworks modernos, como reatividade, diretivas e manipulação de estado, permitindo que o usuário gerencie uma lista de compras de forma dinâmica e interativa.

---

## ⚙️ Funcionalidades

* ✅ Adicionar produtos à lista
* ✅ Remover itens individualmente
* ✅ Contador automático de itens
* ✅ Validação para evitar itens vazios
* ✅ Renderização dinâmica da interface

---

## 🧠 Conceitos Aplicados

* 🔄 Reatividade com `ref()`
* 🔗 Two-Way Data Binding com `v-model`
* 🔁 Renderização de listas com `v-for`
* 🔀 Condicionais com `v-if` e `v-else`
* 🧮 Manipulação de arrays (push, splice)

---

## 🖥️ Interface

* Campo para digitação de produtos
* Botão para adicionar item
* Lista dinâmica de compras
* Botão de remoção em cada item
* Mensagem quando a lista estiver vazia

## 🚀 Como Executar o Projeto

### 1. Clone o repositório

```bash
git clone https://github.com/AndreFerreiraHashimoto/Gestao_de_Compras.git
```

### 2. Acesse a pasta do projeto

```bash
cd gestao-compras
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Execute o projeto

```bash
npm run dev
```

### 5. Acesse no navegador

```
http://localhost:5173
```
