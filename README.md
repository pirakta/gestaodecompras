🛒 Sistema de Gestão de Compras

Este projeto tem como objetivo facilitar o controle e gerenciamento de compras, permitindo registrar pedidos, fornecedores, produtos e acompanhar todo o fluxo de aquisição de forma organizada e eficiente.

📌 Funcionalidades
Cadastro de produtos
Cadastro de fornecedores
Registro de pedidos de compra
Controle de status dos pedidos (pendente, aprovado, entregue, cancelado)
Histórico de compras
Relatórios básicos (por período, fornecedor, produto)
Controle de custos
